Git & GitHub Handy Note

A quick reference guide for all essential Git commands

  • git status – Check the status of the repository.
  • git log --oneline – View commit history in one line.
  • git diff – Show unstaged changes.

  • git init – Initialize a new repository.
  • git clone <repo-url> – Clone a repository.
  • git commit -m "message" – Commit changes with a message.

  • git branch – List branches.
  • git checkout -b <branch> – Create & switch to a new branch.
  • git merge <branch> – Merge a branch.

  • git branch -d <branch> – Delete a branch.
  • git tag -d <tag> – Delete a local tag.
  • git reset --hard HEAD~1 – Delete last commit.
